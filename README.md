# 🕒 Analog Watch

A simple and responsive analog watch built using HTML, CSS, and JavaScript. This project displays the current time with smooth movement of hour, minute, and second hands. It is a beginner-friendly project to understand DOM manipulation and real-time updates.

---

## 🚀 Features

* ⏰ Real-time analog clock
* 🎨 Clean and modern UI
* ⚡ Smooth hand rotation
* 💻 Beginner-friendly code

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
